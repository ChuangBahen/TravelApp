## 📅 Day 1｜4月10日（五）抵達東京・上野

**🛫 航班資訊：** 台灣桃園 10:40 → 日本成田 15:05

### 行程安排

| 時間 | 行程 |
|:---:|:---|
| 15:05 | 抵達成田機場第一航廈 |
| 15:30-16:00 | 入境、領取行李、購買 Skyliner 車票 |
| 16:30 | 搭乘 Skyliner 前往上野（約41分鐘） |
| 17:15 | 抵達京成上野站 |
| 17:30 | 飯店 Check-in、休息整理 |
| 18:30 | 晚餐：上野周邊 |

### 🍽️ 晚餐推薦

| 餐廳 | 特色 | 素食選項 |
|:---|:---|:---|
| **韻松亭** | 百年歷史、精緻日式料理 | 有素食套餐 |
| **上野精養軒** | 老牌洋食、咖哩飯 | 可詢問素食餐點 |
| **guruatsu（備案）** | 蔬食專門店、30種以上蔬菜 | 全素食 |

### 🏨 住宿

**三井花園飯店上野（Mitsui Garden Hotel Ueno）**
- 地址：東京都台東区東上野3-19-7
- 交通：京成上野站步行約5分鐘
- 特色：大廳設有熊貓裝飾、大浴場、家庭房型選擇多
- 備案：上野HOTEL MYSTAYS / 上野DORMY INN

### 🗺️ 交通路線

```
成田機場 → 京成上野站（Skyliner 約41分鐘）→ 飯店（步行5分鐘）
```

### 🐼 上野動物園資訊

- 營業時間：9:30～17:00（最後入園 16:00）
- 門票：成人 600日圓｜65歲以上 300日圓｜國中生 200日圓｜**小學以下免費**
- 必看：熊貓「曉曉」「蕾蕾」、鯨頭鸛、北極熊、企鵝
- 親子設施：嬰兒車租借（500日圓/天）、兒童動物園互動區

### 🍽️ 午餐推薦

| 餐廳 | 特色 | 價位 |
|:---|:---|:---:|
| **園內 Saruyama Kitchen** | 可邊看猴山邊用餐、有熊貓造型餐點 | ¥800-1200 |
| **上野公園野餐** | 便利商店購買便當，享受春日櫻花 | ¥500-800 |

### 🛍️ 購物行程

**松坂屋上野店**（步行約8分鐘）
- 營業時間：10:00～20:00

**阿美橫町商店街**（步行約5分鐘）
- 特色：藥妝、零食、小吃攤，熱鬧有趣適合孩童

### 🍽️ 晚餐推薦

|:---|:---|:---|
| **伊豆榮** | 300年鰻魚飯名店 | 需事先確認 |
| **上野 藪そば** | 手打蕎麥麵 | 有蔬菜天婦羅蕎麥 |

### 🏨 住宿
